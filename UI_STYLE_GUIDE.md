# Guía de Estilo y UI: HorariosQuedate

## Tipografía
- **Fuente Principal:** 'Inter' o 'Geist Sans' (vía @fontsource/inter).
- **Características:** Sans-serif, limpia, optimizada para legibilidad en pantallas con datos densos.
